# Python Calculator

A simple calculator built with Python using Object-Oriented Programming (OOP).

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Power
* Factorial

## Technologies Used

* Python 3
* OOP
* Static Methods
## Example Output

```text id="yo4vqk"
Enter a number: 6
Enter another number: 3
Enter a number to find its factorial: 5
Enter the (base, exponent) : 3 3

   **Results:**   
**the sum of 6 and 3 is: 9
**the difference of 6 and 3 is: 3
**the multiplication of 6 and 3 is: 18
**the division of 6 and 3 is: 2.0
**the power of 3 to the 3 is: 27
**the factorial of 5 is: 120
```
